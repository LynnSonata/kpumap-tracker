# 📍 KPU Surrey Library Distance Tracker

# Overview
This is a web-based application built for the INFO 4235 S11
It dynamically tracks a user's location using browser geolocation and visually displays their real time distance in kilometers to the Kwantlen Polytechnic University (KPU) Surrey Library

# Features
* Live GPS Tracking: Utilises the HTML5 Geolocation API
* Interactive Map: Renders a responsive map using Leaflet.js and OpenStreetMap
* Dynamic Distance Calculation: Uses the Haversine formula written in JavaScript to calculate the exact distance between the user's live coordinates and the KPU Surrey Library

# How to Run Locally
Because this project was built using vanilla web technologies with no backend framework required, running it is incredibly simple:
1. Clone or download this repository
2. Open the "index.html" file directly in any modern web browser
3. When prompted by the browser, click "Allow" for location access so the map can fetch your GPS coordinates
